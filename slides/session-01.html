📚 DBT là gì?

💬 Định nghĩa

DBT (Data Build Tool) là công cụ transformation-as-code cho data warehouse.

DBT cho phép bạn viết SQL SELECT statements để transform dữ liệu, sau đó tự động hóa việc tạo tables/views trong data warehouse.

🔑 Core Concept

DBT follows TDD (Transform-Driven Development) cho data

Mọi transformation đều được version control, test, và document như code.

⚡ DBT vs Glue

DBT tập trung vào transformation, không phải compute

Glue = Spark-based ETL (compute + transform)
DBT = SQL-based transform (chỉ transform, dùng warehouse có sẵn)

🎯 Use Cases

Perfect cho analytics engineering

✓ Data cleaning & deduplication
✓ Business logic aggregation
✓ Data modeling (star schema)
✓ Data quality testing

🎯 Tại sao DBT thay thế Glue?

Feature AWS Glue DBT Core
Learning Curve PySpark/Scala required SQL (ai cũng biết!)
Development Speed Chậm (write code → test → debug) Nhanh (viết SQL → run → see results)
Testing Manual (viết test code) Built-in (declarative tests)
Documentation External (Confluence, README) Auto-generated từ YAML
Version Control Có (Git integration) Native (file-based)
Cost Pay per DPU-hour Free (open source)
Vendor Lock-in Cao (AWS ecosystem) Thấp (works everywhere)
Debugging CloudWatch logs Local execution + clear errors

🛠️ Setup Environment

1

Yêu cầu hệ thống

  • Python 3.9+
  • pip hoặc conda
  • Truy cập S3 (AWS credentials)
  • Truy cập PostgreSQL (RDS endpoint)
  • EC2 instance (hoặc local machine để test)
2

Cài đặt DBT Core

# Tạo virtual environment (recommended)
python -m venv dbt-venv
source dbt-venv/bin/activate  # Linux/Mac
# hoặc
dbt-venv\Scripts\activate  # Windows

# Cài đặt DBT với PostgreSQL adapter
pip install dbt-postgresql

# Cài đặt thêm S3 plugin (nếu cần load từ S3)
pip install dbt-s3
3

Khởi tạo DBT Project

# Tạo project mới
dbt init dbt-shopee-tiktok

# Chọn database adapter: postgresql
# Nhập connection details:
#   - host: your-rds-endpoint.rds.amazonaws.com
#   - port: 5432
#   - user: your_username
#   - password: ******
#   - database: analytics
#   - schema: dbt_output
4

Configure profiles.yml

# ~/.dbt/profiles.yml
dbt-shopee-tiktok:
  outputs:
    dev:
      type: postgres
      host: your-rds-endpoint.rds.amazonaws.com
      port: 5432
      user: your_username
      password: your_password
      dbname: analytics
      schema: dbt_output
      threads: 4
    prod:
      type: postgres
      host: your-rds-endpoint.rds.amazonaws.com
      port: 5432
      user: prod_user
      password: prod_password
      dbname: analytics
      schema: analytics
      threads: 8
  target: dev

⚠️ Lưu ý: profiles.yml KHÔNG được commit vào Git!

5

Test Connection

# Test kết nối
dbt debug

# Chạy thử models
dbt run

# Chạy tests
dbt test

# Generate docs
dbt docs generate
dbt docs serve
